# 머강대 학사행정 AI 클라우드 인프라 - AWS 최적화(5만원 이하 실속형) 아키텍처 명세서

너무 초라해 보이지 않으면서도(포트폴리오용), AWS의 핵심 기능(서버리스, 스토리지, 컨테이너, DB, AI)을 골고루 분산 활용하여 예산(5만원) 내에 맞춘 **'실속형 10개 리소스 아키텍처'**입니다.

이 구성은 **각 기술 스택의 목적을 뚜렷하게 보여주는 증적용**으로 매우 적합합니다.

---

## 🏗️ 예산 5만원 이하 방어 + 풍부한 포트폴리오용 10개 리소스 세트

### 1️⃣ 네트워크 (Network) - 트래픽 진입로
포트폴리오상 "아, 이 사람이 AWS 네트워크 격리 개념을 아는구나"를 보여주기 위해 프라이빗(Private) 망 흉내라도 내야 합니다. 하지만 고비용의 NAT Gateway는 뺍니다.
1.  **VPC (Virtual Private Cloud)**: 전체 클라우드 리소스를 담는 우리만의 망.
2.  **Subnets (Public / Private)**: 서버(EC2)와 DB(RDS)를 격리하는 용도. (NAT 없이 프라이빗을 흉내냅니다)
3.  **IGW (Internet Gateway)**: 퍼블릭 서브넷의 외부 연결 지원.
4.  **ALB (Application Load Balancer)**: 트래픽을 분산하는 관문. k8s 내부 로드밸런서와 연동하여 외부 트래픽을 처리합니다. **(포트폴리오 필수)**

### 2️⃣ 컴퓨팅 / 컨테이너 오케스트레이션 (Compute & Containers)
가장 돈이 많이 드는 EKS 대신 단일 인스턴스로 해결하되, "Docker와 k8s 생태계를 다룹니다"를 증명합니다.
5.  **EC2 (Ubuntu, `t3.medium`)**: K3s(Docker+Kubernetes)를 띄우는 핵심 머신. '스팟 인스턴스(Spot)'로 띄워서 비용을 극한으로 줄입니다.
6.  **ECR (Elastic Container Registry)**: Github Actions CI/CD 파이프라인에서 빌드한 도커 이미지를 안전하게 관리.

### 3️⃣ 데이터베이스 시스템 (Database & DB)
단순 연습용이 아니라, 진짜 운영(Prod) 환경처럼 관리형 데이터베이스에 연결하는 아키텍처를 보여줍니다.
7.  **RDS (PostgreSQL, `db.t3.micro`)**: 애플리케이션 회원, 수강, 스터디 그룹 데이터를 안전하게 보관. 추가적으로 **대기열(Waitlist)** 트랜잭션 관리와 동시성 락(Concurrency Lock) 제어 명세를 반영한 정규화된 스키마. **프리티어** 버전을 적극 활용하여 월 요금을 0에 가깝게 방어합니다.
    *   **Bastion Host (경비실 서버) 터널링 보안:** RDS는 프라이빗 서브넷에 완전히 격리되어 외부 접속이 차단되어 있습니다. 파이썬 백엔드는 EC2 우회 터널(SSH Tunnel - Local Port Forwarding)을 뚫어 안전하게 데이터베이스에 접근합니다.

### 4️⃣ 스토리지 (Storage) - 문서 파일 관리
AI RAG(검색 증강 생성) 파이프라인의 원본이 되는 강의자료 PDF 등을 저장합니다.
8.  **S3 (사전서명된 URL 활용)**: 웹사이트(Frontend)에서 사용자가 파일을 올릴 때, 백엔드 컨테이너 램(RAM)을 안 잡아먹도록 S3로 직접(Direct Upload) 올리는 고급 기법 증명용.

### 5️⃣ AI 파이프라인 및 백그라운드 태스크 (AI API)
"서버리스 컴퓨팅과 LLM AI"라는 미래지향적 키워드를 이력서에 넣기 위한 핵심 무기입니다.
9.  **AWS Lambda (비동기 처리)**: 사용자가 S3에 문서를 업로드하는 순간 발동! 문서를 텍스트로 잘게 쪼개는 작업(Chunking)을 메인 서버가 아니라 서버리스가 대신 처리합니다.
10. **Amazon Bedrock (LLM 모델 접속)**: GPT 대용으로 쓰는 AWS 자체 AI API. 과제/채점 증적용으로 과금(종량제)을 철저히 모니터링하며 사용합니다.

## 💰 예상 비용 시뮬레이션 (Total: 약 $20 ~ $25, 약 3만원 대 초반 방어)
*   **ALB**: 월 $16
*   **EC2 (`t3.medium` 스팟 1대)**: 월 약 $10 ~ $12
*   **RDS (Free Tier)**: 요건만 맞으면 월 0원. 용량 20GB 초과 시 초과분 지불 (개발 기간엔 안 넘음).
*   **S3 & ECR & Lambda**: 종량제 (거의 1~2달러 수준)
*   **Bedrock (AI API)**: 테스트 횟수에 비례 (약 2~3달러)
*   **총합**: 한 달 쭉 켜놔도 3~4만원 수준 방어 성공!
